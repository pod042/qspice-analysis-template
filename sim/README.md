# Simulation examples

## Overview
This directory contains example simulations for QSPICE analysis.

### Example 1: Simple Proof of Concept
A basic simulation demonstrating fundamental circuit behavior and validation principles. Ideal for understanding simulation setup and basic analysis workflows.

### Example 2: Totem-Pole Bridgeless Boost PFC Converter
A comprehensive open-loop simulation of a totem-pole bridgeless boost Power Factor Correction (PFC) converter. This example serves as a template for developing and testing control algorithms.
